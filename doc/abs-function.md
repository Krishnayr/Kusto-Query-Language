# abs()

Calculates the absolute value of the input.

## Syntax

`abs(`*x*`)`

## Parameters

| Name | Type | Required | Description |
| -- | -- | -- | -- |
| *x* | int, real, or timespan | &check; | The value to make absolute. |

## Returns

* Absolute value of x.
